# Steps Directory

**Purpose:** This directory contains minimal, command-focused installation steps. These are lightweight versions of modules, containing primarily commands with minimal explanatory text.

---

## What are Steps?

Steps are **command-focused guides** that include:
- ✅ Essential commands only
- ✅ Minimal explanatory text
- ✅ Quick reference format
- ✅ For experienced users
- ✅ Lightweight and fast to read

---

## Steps vs Modules

**Steps** (this directory):
- Commands only
- Minimal text
- Quick reference
- For experienced users
- Fast execution

**Modules** (`../modules/` directory):
- Detailed explanations
- Troubleshooting included
- Educational content
- For all skill levels
- Comprehensive guides

---

## When to Use Steps

Use steps when:
- ✅ You're experienced with Arch Linux
- ✅ You know what each command does
- ✅ You want quick reference
- ✅ You're following a known procedure
- ✅ You don't need explanations

Use modules when:
- ✅ You're learning Arch Linux
- ✅ You want to understand what you're doing
- ✅ You need troubleshooting help
- ✅ You want detailed explanations
- ✅ You're doing something for the first time
